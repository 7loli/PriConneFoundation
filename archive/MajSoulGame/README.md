> 由于某人懒得抽，幸运奖跟点炮奖 ~~没发~~ 并入下一届奖金池

**重要**：
1. 四强奖励仅限于**游戏内消费，不可折现**，不限制游戏内使用方式（可分开购买辉玉、御守、服装券，建议优先选择首充双倍）；
2. 幸运奖从四强以外**所有参赛人员**中随机抽取，可选择游戏内御守/月卡 或 折现 20 软妹币；
3. 点炮奖为新追加奖项，16 强比赛中**放铳点数最高者**才能获得此荣誉，不可与幸运奖叠加。
